## О содержании работы

В статье даётся аппаратно-ориентированное описание семейства числовых форматов GoldenFloat, порождаемого единым замкнутым правилом разбиения разрядной сетки между полем порядка и полем мантиссы: для каждой полной разрядности N ≥ 4 ширина поля порядка равна e = round((N−1)/φ²). Представлены три воспроизводимых артефакта:

1. открытый многоразрядный RTL-генератор (GF4–GF256) с дифференциальной прогонкой в режиме непрерывной интеграции относительно корректно округлённого эталона;
2. целочисленный тракт аккумулятора, точный по числам Люка, проверенный с точностью до 500 знаков для n = 1…256;
3. кодек GF16 на FPGA, проходящий тестовое окружение 35/35 на частоте 323 МГц (Artix-7, Xilinx XC7A35T).

Правило позиционируется рядом с posit (Posit Standard 2022), takum, OCP-MX и черновиком IEEE P3109. Работа не содержит заявлений о превосходстве по точности: ключевой тезис о широте/согласованности зафиксирован как открытая гипотеза с предрегистрированным путём фальсификации (реестр FL-002), что повышает прозрачность для рецензента. Историческая часть введения связывает работу с отечественной линией троичных вычислений (ЭВМ «Сетунь», Н.П. Брусенцов, МГУ, 1958).
