# Русская версия научной статьи 2 — каталог численных форматов

## Файл
`statya2_ru.docx` — полная русскоязычная версия статьи (20 страниц, Times New Roman 12 pt).

## О чём статья
Статья описывает **вендор-нейтральный справочник из 84 численных форматов**, организованных в **13 кластеров**, с набором из **шести битоточных пакетов соответствия (conformance packs)**: GF16, элемент MXFP4, BF16, FP8 E4M3, FP8 E5M2 и блочный масштаб E8M0. Каждый пакет — самодостаточный документ JSON с отпечатком SHA-256, общей схемой строк и якорным вектором, кодирующим значение 3,0 через тождество **φ² + 1/φ² = 3**.

Ключевые элементы:
- **Кросс-уолк к IEEE P3109 v3.2.0** — сопоставление каждого пакета с конфигурируемыми форматами стандарта.
- **Перекрёстная проверка** относительно эталонного оракула `ml_dtypes` 0.5.4 (Google/JAX); все расхождения документируются явно (например, интерпретационный разрыв при переполнении FP8 E4M3).
- **Два задокументированных интерпретационных разрыва**: политика переполнения FP8 E4M3 (SatMax против OvfNaN) и блочная структура 4-битных форматов (MXFP4 против NVFP4).
- Позиционирование как **заполнение реестра**: без новых форматов, без утверждений о точности моделей, без сравнения вендоров.
