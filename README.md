# Trinity Papers — русские версии для журналов ВАК

Русскоязычные версии научных статей Trinity (gHashTag / Дмитрий Васильев) для публикации в российских журналах перечня ВАК. Готовятся под защиту кандидатской диссертации по специальности **1.2.1 «Искусственный интеллект и машинное обучение»**.

Спутник репозиториев: [`gHashTag/t27`](https://github.com/gHashTag/t27), [`gHashTag/goldenfloat-preprint`](https://github.com/gHashTag/goldenfloat-preprint), [`gHashTag/paper3-methodology`](https://github.com/gHashTag/paper3-methodology).

## Статья 1 — GoldenFloat

- Папка: [`paper1-goldenfloat/`](paper1-goldenfloat/)
- Формат: **LaTeX** (`main_ru.tex`) + скомпилированный `main_ru.pdf` (22 стр.)
- Целевой журнал: **«Программирование» / Programming and Computer Software** (ИСП РАН, Pleiades/Springer) — категория **К-1**, индексируется в Scopus.
- Тема: семейство форматов с плавающей точкой GoldenFloat (статическое разбиение, выведенное из φ), целочисленный аккумулятор на основе тождества Люка, FPGA-реализация GF16.
- Оригинал: arXiv:2606.05017.
- Покрывает work packages WP2 + срез WP1 диссертационного плана.

## Статья 2 — Каталог из 84 численных форматов

- Папка: [`paper2-catalog/`](paper2-catalog/)
- Формат: **Word** (`statya2_ru.docx`, 20 стр.)
- Целевой журнал: **«Искусственный интеллект и принятие решений»** (ФИЦ «Информатика и управление» РАН) — категория **К-1**, специальность 1.2.1.
- Тема: каталог из 84 численных форматов в 13 кластерах с битоточными векторами соответствия, кросс-уолк к IEEE P3109, якорное тождество φ² + 1/φ² = 3.
- Источник: `gHashTag/paper3-methodology`, `gHashTag/t27`.

## Требования ВАК (2026)

Для защиты по техническим/физматнаукам — не менее 2 статей перечня ВАК, минимум одна в категории К-1/К-2 («Белый список» РЦНИ / RSCI / Scopus). Обе статьи выше — К-1, что закрывает требование с запасом. Перед подачей проверять актуальный статус журнала на [journalrank.rcsi.science](https://journalrank.rcsi.science/ru/).
