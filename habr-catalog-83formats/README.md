# Habr: Каталог из 83 числовых форматов (Golden Ruler)

Вторая статья проекта GoldenFloat. Про единый машинно-проверяемый каталог
форматов чисел, статус-метки, якорь 0x47C0 и честный count-drift (84/83/81/77).

Препринт каталога: [arXiv:2606.09686](https://arxiv.org/abs/2606.09686)
